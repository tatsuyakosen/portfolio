クラウド × モバイル × Web UI に特化したフルスタックエンジニア

自己紹介

クラウド・モバイル・Web UI を軸にスキルを広げてきたソフトウェアエンジニアです。
AWS / GCP の複数資格を保持し、CDK・OpenAPIによるサーバーレス構成やFlutter・Reactを用いたUI開発を得意としています。
設計〜構築〜トラブル対応まで一貫して担い、技術面だけでなくプロダクトの成長にも寄与できるエンジニアを目指しています。
